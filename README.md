Como utilizarlo

Bueno, este archivo lo hice por que cada semana transmito una lista de canciones por listen2myradio y siempre tenia que desccargar las canciones y luego reenumerarlas para poder luego crear 
la lista de las canciones para enviarlas a la IA de google para que me generase datos de las mismas.

Solo coloca el archivo en la carpeta donde estan los mp3 y ejecutalo. esto reenumerara todos los archivos en el orden en que fueron descargadas
