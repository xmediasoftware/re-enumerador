How to use it

Well, I created this file because I stream a playlist of songs every week on listen2myradio, and I always had to download the songs and then renumber them to create the playlist to send to Google's AI so it could generate data for them.

Just place the file in the folder where your MP3s are and run it. This will renumber all the files in the order they were downloaded.
